Installation

Deploy using Helm in seconds.

# Add the repository
helm repo add ndots https://hawky-4s-.github.io/k8s-ndots-admission-controller
helm repo update

# Install the chart
helm install k8s-ndots-admission-controller ndots/k8s-ndots-admission-controller \
  --namespace ndots-system \
  --create-namespace
